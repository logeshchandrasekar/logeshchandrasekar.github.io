## About

This is my personal developer portfolio — a single-page site built from scratch with vanilla HTML, CSS, and JavaScript. It covers who I am, what I've built, and what I'm looking for next.

## Sections

- **Intro** — animated avatar, quick pitch, and a downloadable resume
- **About** — background, working style, and what I'm looking for in a role
- **Skills** — visual tech stack grouped by Languages, Frontend & Frameworks, Backend & APIs, Databases, and Tools
- **Projects** — a look at what I've built
- **Education** — academic background
- **Contact** — direct links plus a contact form

## Built With

- HTML5, CSS3, JavaScript — vanilla, no framework or build step
- Google Fonts (Manrope)
- Fully responsive, with breakpoints at 900px, 800px, and 480px
- Deployed via GitHub Pages

## Running Locally

It's a static single-file site, so there's nothing to install or build.

```bash
git clone https://github.com/logeshchandrasekar/logeshchandrasekar.github.io.git
cd logeshchandrasekar.github.io
open index.html   # or just double-click it
```
