Backend / API / Systems

Built around a producer-consumer model with Redis-backed distributed locking, heartbeat-based failure detection, and automatic task reassignment, with a real-time dashboard for cluster health.

Role
Designed and built the full system — task ingestion API, distributed locking layer, heartbeat monitor, and fault-recovery logic.
Tech Stack
Redis, PostgreSQL, React + Tailwind, Docker
